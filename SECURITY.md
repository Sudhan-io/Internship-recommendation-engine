# Security Policy

## Supported Versions
Only the latest major version (`v1.x`) of InternMatch is currently supported with security updates.

## Reporting a Vulnerability

We take the security of InternMatch seriously. If you discover a security vulnerability within the project, please **do not** disclose it publicly via GitHub Issues. 

Instead, please send an email to the project maintainers or use the private GitHub Security Advisory feature if available. 

When reporting a vulnerability, please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.
- Any potential mitigation or solutions you can suggest.

We will acknowledge your report within 48 hours and work with you to understand and resolve the issue. Once resolved, we will publish a security advisory and credit you for the discovery.
